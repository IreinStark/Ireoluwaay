Appealing vs. Unappealing Web Design

This page answers each required question for one website I find appealing and one I find unappealing. It is also styled to print cleanly if a PDF version is needed from the browser.

Appealing Example

Pinterest

URL: https://www.pinterest.com/

a. What is the URL of the website?

https://www.pinterest.com/

b. Does this page use tables? If so, for what purpose?

Pinterest does not use tables for layout or content presentation. The signature masonry grid is built with modern CSS and JavaScript, allowing content cards to flow fluidly into an asymmetrical, visually engaging arrangement.

c. Does this page use CSS? If so, for what purpose?

Yes, extensively. CSS drives the masonry layout, the monochromatic UI palette that lets user content take center stage, smooth hover transitions on pins, responsive behavior across screen sizes, and the overall visual hierarchy of the page.

d. Is this page appealing or unappealing? List three reasons.
  • The fluid, asymmetrical masonry layout feels dynamic and encourages exploration rather than passive scrolling.
  • The monochromatic UI steps back and lets the vibrant user-generated images become the focal point.
  • Smooth hover states and seamless transitions make interactions feel polished and responsive.
e. If this page is unappealing, what would you do to improve it?

This is my appealing example, so the design works well overall. A minor improvement would be giving users more control over grid density so those who prefer a calmer layout can reduce visual busyness.

Unappealing Example

Craigslist

URL: https://www.craigslist.org/

a. What is the URL of the website?

https://www.craigslist.org/

b. Does this page use tables? If so, for what purpose?

Craigslist relies heavily on dense, rigid text columns to organize its category links. The result is a wall of text with little visual separation, making it difficult to scan and navigate at a glance.

c. Does this page use CSS? If so, for what purpose?

The page uses minimal CSS. Styling is largely limited to basic link colors and simple spacing, with no meaningful hierarchy, visual grouping, or responsive design. The default HTML blue hyperlinks are used throughout with little customization.

d. Is this page appealing or unappealing? List three reasons.
  • The dense, rigid text columns offer no visual breathing room, making the page feel overwhelming on first load.
  • Default HTML blue links and sterile backgrounds give the site an unfinished, dated appearance.
  • Harsh, instant page reloads with no transitions make the experience feel abrupt compared to modern web standards.
e. If this page is unappealing, what would you do to improve it?
  • Replace the text column layout with a clean card grid that uses whitespace and icons to separate categories visually.
  • Introduce a consistent color palette and updated typography to replace the default browser styling.
  • Add subtle hover states and smoother page transitions to reduce the jarring feel of navigation.
